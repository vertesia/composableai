# Report Template

This is an example template for generating PDF reports.

## Usage

Use this template with the document generation system to create formatted reports.
The template includes a title page SVG that can be customized with Handlebars variables.

### Available Variables

- `{{title}}` - The report title
- `{{subtitle}}` - Optional subtitle
- `{{author}}` - Author name
- `{{date}}` - Report date
